数据库知识点

参数化

  • sql语句的参数化，可以有效防止sql注入
  • 注意：此处不同于python的字符串格式化，全部使用%s占位
  • 创建test_param.py文件，向学生表中插入一条数据
#coding=utf-8
from pymysql import *

if __name__=='__main__':
    sname = raw_input("请输入学生姓名：")

    try:
        #创建Connection连接
        conn=connect(host='localhost',port=3306,user='root',password='mysql',database='python1',charset='utf8')
        #获得Cursor对象
        cs1=conn.cursor()
        #构造参数列表
        params = [sname]
        #执行insert语句，并返回受影响的行数：添加一条学生数据
        count = cs1.execute('insert into students(name) values(%s)', params)
        #打印受影响的行数
        print count
        #关闭Cursor对象
        cs1.close()
        #提交之前的操作，此处为insert操作
        conn.commit()
    except Exception as e:
        print e
    finally:
        #关闭Connection对象
        conn.close()
